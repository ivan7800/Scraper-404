TRANSPARENCIA

Privacidad, precisión y límites

Modo web

El análisis se ejecuta en tu navegador. Cuando una web no permite CORS, la descarga puede pasar por AllOrigins, CodeTabs o el proxy que configures. Esos servicios reciben la URL. Las direcciones privadas o locales nunca se reenvían a proxies públicos.

Motor local opcional

El servicio de localhost abre la página con Playwright en tu propio equipo. Puede ejecutar JavaScript, hacer scroll, crear capturas o PDF, conservar un perfil de sesión y registrar peticiones XHR/fetch. La comunicación está protegida con una clave aleatoria. HTML, cookies, capturas, base SQLite y perfiles permanecen en tu dispositivo.

Servicios externos opcionales

El rescate SPA web usa r.jina.ai, Wayback Machine consulta archive.org y la vista SERP puede solicitar el favicon a Google. Ollama se admite únicamente en localhost. Ningún servicio se activa sin una acción tuya.

Límites reales

El motor local no rompe CAPTCHA, Cloudflare, paywalls, autenticación ni controles de acceso. La sesión visible existe para que el usuario complete legítimamente esos pasos. El rastreo se limita al mismo origen y a los máximos elegidos. El análisis visual continúa siendo una inferencia y debe revisarse antes de reutilizar un diseño.

Uso responsable

Analiza únicamente contenido público para el que tengas autorización o un propósito legítimo. Respeta términos de uso, propiedad intelectual, privacidad, robots.txt y legislación aplicable.
